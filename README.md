# HUFSIST
#### <키워드 기반 강의 추천 및 학사 알림 안드로이드 어플리케이션>
#### 한국외국어대학교 컴퓨터공학과 졸업 프로젝트
* 진행 기간 : 2019.03 ~ 2019.06
---
* 기능1 : user에 키워드를 입력받아 그에 맞는 강의 추천
* 기능2 : 기존의 강의 검색 시스템보다 더 상세한 필터링 기능
* 기능3 : e-class 및 학교홈페이지 공지사항 등의 새글 푸시 알림
#### 참여 : 백형기, 유재준, 임예찬
